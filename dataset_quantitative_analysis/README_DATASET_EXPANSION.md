## ðŸ—‚ï¸ DATA STRUCTURE

### Variables per Observation

**Identifying:**
- `jurisdiction_id`: Unique code (e.g., "AR_CRIM", "CL_LABOR")
- `jurisdiction_name`: Full name
- `legal_domain`: Criminal | Labor | Constitutional
- `region`: Latin_America | Europe | Common_Law | Asia | Africa
- `legal_system`: Civil_Law | Common_Law | Mixed

**Independent Variable: CSI (0.0-1.0)**
- `citation_endogamy_rate`: % intra-tradition citations
- `sacred_language_freq`: Frequency of absolutist terms
- `empirical_resistance`: Cases of evidence rejection
- `orthodoxy_requirement`: Ideological conformity for membership
- `exclusion_rate`: Rate of heterodox exclusion
- `journal_concentration`: Herfindahl index of editorial control
- `gatekeeper_concentration`: % forums controlled by orthodoxy
- **Composite:** `csi_total` = weighted average

**Dependent Variable: REI (0.0-1.0)**
- `reforms_proposed`: Number of reforms (2010-2024)
- `reforms_implemented`: Number successfully enacted
- `implementation_rate`: Proportion implemented
- `alignment_score_mean`: Average alignment with stated objectives
- `evaluation_mechanisms`: Presence of evidence-based review
- `evidence_based_revisions`: Rate of modifications based on evidence
- `reversal_rate`: 1 - (proportion reversed within 5 years)
- **Composite:** `rei_total` = weighted average

**Control Variables:**
- `gdp_per_capita`: World Bank (2020)
- `population`: UN data (2020)
- `democracy_score`: V-Dem Liberal Democracy Index (2020)
- `rule_of_law_score`: World Bank Governance Indicators (2020)
- `ideological_orientation`: Left | Center | Right | Mixed

**Metadata:**
- `notes`: Qualitative observations
- `sources`: Full bibliography
- `confidence_level`: High | Medium | Low
- `coder_notes`: Decision justifications

## ðŸ”¬ METHODOLOGY

### Data Collection Protocol (3 hours per observation)

**Hour 1: CSI - Endogamy and Control**
1. Identify top 3 journals in domain (Google Scholar, SciELO)
2. Sample 10 articles (not 30, for efficiency)
3. Code citations: endogamous vs. exogamous
4. Identify editorial board composition
5. Calculate concentration indices

**Hour 2: CSI - Sacralization and Costly Signaling**
1. Select 5 influential texts (treatises, highly-cited articles)
2. NLP analysis: frequency of sacralizing terms
3. Search for documented controversies (exclusion cases)
4. Code orthodoxy requirements (hiring patterns, associations)

**Hour 3: REI - Reforms and Outcomes**
1. Identify 5-10 major reforms (last 15 years)
2. Classify: proposed/enacted/reversed
3. Find 1-2 key indicators per reform (pre-post data)
4. Calculate implementation and alignment scores

**Quality Standards:**
- Minimum 3 independent sources per variable
- Cross-reference with secondary literature when available
- Flag low-confidence variables for sensitivity analysis
- Document all coding decisions in supplementary notes
